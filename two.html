justify-content: justify-content in Flexbox is used to control the alignment of items along the main axis of the flex container. Values include flex-start (items align to the start of the main axis), flex-end (items align to the end), center (items center along the main axis), space-between (equal spacing between items, excluding the first and last), space-around (equal spacing around items, including before the first and after the last), and space-evenly (equal spacing around all items).

align-items: align-items determines how items are aligned along the cross-axis of the flex container. Values include flex-start (items align at the start of the cross axis), flex-end (items align at the end), center (items center along the cross axis), baseline (items align to their baselines), and stretch (items are stretched to fill the container).

flex-direction: flex-direction sets the main axis of the flex container, defining the direction in which flex items are placed. Values include row (items placed in a row, the main axis is horizontal), column (items placed in a column, the main axis is vertical), row-reverse (items placed in a row but in the opposite order), and column-reverse (items placed in a column but in the opposite order).

flex-wrap: flex-wrap determines whether flex items are forced onto a single line or can wrap onto multiple lines within the flex container. Values include nowrap (all items are on one line), wrap (items wrap onto additional lines if needed), and wrap-reverse (items wrap onto additional lines in the opposite order).
